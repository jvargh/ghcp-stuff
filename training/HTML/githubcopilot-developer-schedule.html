GitHub Copilot Developer

    • Code review
    • Copilot Spaces
  • Demo: Features Demo (chat commands, generate/explain code, review + commit)

11:00-12:00 | Presentation + Demo: Engineering Practices Demo

  • Planning: brainstorm and scope using Copilot Chat and Spaces
  • Code creation, testing, and debugging

    • Chat modes and when to use each
  • Demo: Engineering Practices (inspect Copilot context, export/import chats, shared prompts)

12:00-1:00 | Lunch

1:00-2:00 | Presentation + Demo: Customize Copilot Demo

  • Customize Copilot Chat responses to fit preferences and requirements
  • Custom instructions, prompt files, and custom chat modes
  • Model selection and premium request considerations
  • Demo: Customize Copilot (model switching, prompt files, custom instructions, MCP setup)

2:00-2:45 | Presentation + Demo: Copilot Spaces Demo

  • Context handling and best uses (onboarding, system knowledge, style guides)

2:45-3:45 | Presentation + Demo: Coding Agent Demo
