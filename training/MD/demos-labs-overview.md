# Demos Folder High-Level Overview

This overview summarizes the demo guides in the Photo Gallery & Portfolio repo demos folder. The sequence is designed to move from foundational Copilot features to advanced collaboration and automation.

## At a Glance

*   Audience: Developers learning GitHub Copilot features in a Next.js photo gallery app
*   Format: Step-by-step exercises with prompts, expected results, and checklists
*   Flow: Core features -> engineering practices -> customization -> collaboration -> coding agent

## Demo Sequence

### 1) Features Demo

Focus: Copilot fundamentals and workflow basics.  
Highlights:

*   Discover chat commands and slash commands
*   Generate and explain code
*   Review and commit AI-generated changes

### 2) Engineering Practices Demo

Focus: Team-ready Copilot usage and transparency.  
Highlights:

*   Inspect Copilot decision context and prompts
*   Export and import chat conversations
*   Manage shared prompts and conversation knowledge

### 3) Customize Copilot Demo

Focus: Personalization and advanced configuration.  
Highlights:

*   Monitor premium usage and switch models
*   Use prompt files and chat modes
*   Apply custom instructions and MCP server setup

### 4) Copilot Spaces Demo

Focus: Collaborative, goal-based workspaces.  
Highlights:

*   Create spaces with instructions and context
*   Two tracks: security assessment or documentation design
*   Use shared context to solve focused problems

### 5) Coding Agent Demo

Focus: Copilot as an autonomous coding agent.  
Highlights:

*   Assign issues to Copilot
*   Review PRs and session details
*   Practice iterative review and merge workflows

## Workshop Labs (Open-Ended Build)

Focus: Apply Copilot skills to a self-directed build and reflect on outcomes.
Highlights:

*   Choose a project idea (beginner to expert tiers) and build for one hour
*   Set up a new repo, clone locally, and iterate with Copilot help
*   Use Chat commands (for example, `/help`, `@workspace`) when stuck
*   Showcase what you built, how Copilot helped, and lessons learned

## Outcomes

*   Hands-on familiarity with Copilot modes and commands
*   Comfort with team collaboration tools (shared chats, prompts, spaces)
*   Confidence in evaluating and integrating AI-generated changes
*   Practice taking an idea from concept to demo-ready prototype

## Suggested Use

*   Follow the sequence in order
*   Discuss results and compare outputs between participants
*   Capture takeaways and reuse prompt patterns in team workflows
